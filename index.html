Methodology

How this was built.

Data Sources

NOAA GHCN Daily — Central Park station (USW00094728). Historical seasonal totals 1938–2026. The canonical NYC weather station.

CoCoRaHS — Fetched via CoCoRaHS-Download-Tool pip package. Daily NewSnowDepth readings across NY state, filtered to NYC bounding box.

NYC Open Data — Neighborhood Tabulation Areas (NTAs) 2020 boundaries. 262 neighborhoods across 5 boroughs.

Spatial Analysis

Interpolation — When multiple CoCoRaHS stations available, Inverse Distance Weighting (IDW, power=2, k=8) from station locations to NTA centroids.

Borough Aggregation — NTA boundaries dissolved by borough using geopandas. Mean and max snowfall computed per borough per storm.

Pipeline — Google Colab notebook. Full source available on GitHub.
